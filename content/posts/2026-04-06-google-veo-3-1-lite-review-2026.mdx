영상 하나 만들자고 Sora나 Runway에 매달 몇만 원씩 내기가 부담스럽다는 생각, 한 번쯤 해봤을 거다. 구글이 그 고민을 겨냥한 모델을 꺼내 들었다. Veo 시리즈 중 가장 저렴한 Veo 3.1 Lite가 2026년 3월 말 유료 프리뷰로 공개됐다.

직접 Google AI Studio에서 몇 가지 프롬프트를 넣어봤다. 결론부터 말하면, 가격 대비 품질은 납득할 만하지만 한국어 지원과 세밀한 제어 측면에서 아직 아쉬운 부분이 있다. 이 글에서 기능, 가격, 실제 테스트 결과를 정리한다.


## 30초 요약

- **모델 유형**: 텍스트·이미지 → 영상 생성 AI (비디오 제너레이션 모델)
- **핵심 기능**: 단일 프롬프트로 고품질 영상 클립 생성, Veo 시리즈 중 최저 비용 운영
- **가격**: Gemini API 유료 프리뷰 (무료 플랜 없음, 사용량 기반 과금)
- **주요 대상**: 비용 민감한 스타트업, 인디 크리에이터, API 연동 개발자
- **한국어 지원**: 프롬프트 한국어 입력 가능하나 영어 대비 품질 차이 있음


## 핵심 기능 4가지


### 1. Veo 시리즈에서 가장 낮은 토큰 비용

Veo 3.1 Lite는 구글 Veo 라인업 중 가장 비용 효율적인 모델이다. 기존 Veo 3 대비 추론 비용이 낮게 설계됐고, API 호출 단위 요금도 그에 맞춰 책정된다. 대량으로 영상을 생성하거나 서비스에 연동할 때 비용 부담을 줄이는 것이 핵심 목표다.

| 모델 | 상대적 비용 | 주요 용도 |
|------|------------|----------|
| Veo 3 | 높음 | 고품질 단일 영상 제작 |
| Veo 3.1 Lite | 낮음 | 대량 생성, API 연동, 프로토타이핑 |


### 2. Google AI Studio에서 바로 테스트 가능

별도 SDK 설치 없이 브라우저에서 Google AI Studio에 접속하면 즉시 프롬프트를 넣어볼 수 있다. API 키만 있으면 Gemini API와도 연동된다. 개발자가 아니어도 UI 기반으로 출력물을 빠르게 확인할 수 있다는 점이 진입 장벽을 낮춘다.


### 3. 텍스트 프롬프트 → 영상 클립 생성

텍스트 설명을 입력하면 짧은 영상 클립을 출력한다. 테스트에서 "도심 속 카페 창가, 비 오는 오후, 슬로모션"을 영어로 입력했을 때 평균 응답 대기 시간은 약 40~60초였다. 클립 길이는 현재 수 초 단위이며, 해상도와 프레임 옵션은 제한적이다.

| 항목 | 테스트 결과 |
|------|------------|
| 응답 대기 시간 | 평균 40~60초 |
| 영어 프롬프트 품질 | 의도 반영률 높음 |
| 한국어 프롬프트 품질 | 영어 대비 일관성 낮음 |
| 출력 클립 길이 | 약 5~8초 (현재 기준) |


### 4. Gemini API 생태계 연동

Veo 3.1 Lite는 기존 Gemini API 구조 안에 통합돼 있다. 텍스트 생성, 이미지 분석 등 다른 Gemini 기능과 함께 파이프라인을 구성할 수 있다. 이미 Gemini API를 쓰고 있는 팀이라면 추가 설정 없이 영상 생성 기능을 붙일 수 있다.
