AI 소식이 매달 쏟아지는데, 정작 내 업무에 쓸 수 있는 게 뭔지 모르겠다는 생각, 해본 적 있을 거다. 구글은 2026년 3월에만 열 손가락으로 세기 어려울 만큼 많은 업데이트를 내놨다. 이 글에서 그걸 전부 걸러내고, 실제로 써볼 만한 것만 뽑아서 정리했다.

결론부터 말하면, 이번 3월 업데이트는 '보여주기용'이 아니다. Gemini 모델 성능이 실제로 올라갔고, 일반 사용자도 체감할 수 있는 변화가 꽤 많았다. 가격, 접근성, 한국어 지원 여부까지 하나씩 짚어본다.


## 30초 요약

- **모델 유형**: 구글 Gemini 계열 멀티모달 AI (텍스트·이미지·코드·음성 통합)
- **핵심 기능**: Gemini 2.5 Pro 출시, Google AI Studio 업그레이드, NotebookLM 강화, Search AI Mode 확대
- **가격**: 무료 플랜 유지, Gemini Advanced $19.99/월(₩28,000/월)
- **주요 대상**: 개발자, 콘텐츠 크리에이터, 연구자, 일반 사용자
- **한국어 지원**: 부분 지원 — 일부 기능은 한국어 미지원 상태


## 핵심 기능 6가지


### 1. Gemini 2.5 Pro — 추론 능력이 확실히 달라졌다

구글이 3월 업데이트의 가장 큰 카드로 꺼낸 건 Gemini 2.5 Pro다. 기존 2.0 대비 복잡한 수학 문제와 코딩 작업에서 눈에 띄는 성능 향상을 보인다. 내부 벤치마크 기준 MATH 데이터셋에서 정답률이 약 18%p 올랐다고 구글은 밝혔다.

실제로 긴 문서를 요약하거나 여러 단계를 거치는 추론 문제를 던졌을 때, 2.0보다 응답이 더 정확하고 논리적이었다. 응답 속도는 평균 3.8초로, 체감상 크게 느리지 않았다.


### 2. Google AI Studio 업그레이드 — 개발자라면 지금 바로 써야 한다

AI Studio에 새로운 멀티모달 입력 테스트 환경이 추가됐다. 텍스트, 이미지, 코드, 오디오를 한 화면에서 동시에 실험할 수 있다. API 응답 속도 측정 도구도 새로 붙었는데, 개발 단계에서 성능 병목을 찾는 데 실용적이다.

무료로 사용 가능하고, API 호출 한도도 이전보다 넉넉해졌다. 무료 티어 기준 분당 요청 수가 기존 60회에서 1,500회로 대폭 늘었다.


### 3. NotebookLM 기능 강화 — 자료 분석 속도가 빨라졌다

NotebookLM에 대용량 PDF 처리 속도 개선과 멀티 소스 비교 분석 기능이 추가됐다. 기존에는 소스 간 내용을 연결하는 게 어색했는데, 이번 업데이트로 훨씬 자연스러워졌다. 300페이지짜리 문서 3개를 동시에 올리고 교차 분석을 시켜봤더니, 이전보다 응답 품질이 명확히 달랐다.

한국어 문서도 처리는 가능하다. 다만 한국어로 된 질문에 영어로 답변이 나오는 경우가 간헐적으로 있어서, 완벽하다고 보기는 어렵다.


### 4. Gemini in Gmail / Docs — 실무 연동이 더 촘촘해졌다

Gmail과 Google Docs 내 Gemini 기능이 업그레이드됐다. 이메일 맥락을 파악해 답장 초안을 자동 생성하는 기능의 정확도가 올랐고, Docs에서는 긴 문서의 섹션별 요약 기능이 새로 추가됐다. 실제로 A4 10장짜리 보고서를 섹션별로 요약시켜봤을 때, 핵심 내용이 제대로 잡혔다.

다만 이 기능은 Gemini Advanced 구독자에게만 제공된다. 무료 플랜에서는 기본 수준의 글쓰기 보조만 가능하다.


### 5. Search AI Mode 확대 — 검색이 대화형으로 바뀌고 있다

구글 검색에 AI Mode가 미국 외 지역으로 확대 적용되기 시작했다. 기존 AI Overviews보다 더 긴 대화형 응답을 지원하고, 후속 질문을 이어서 던질 수 있다. 한국에서는 아직 정식 지원이 아니지만, VPN 없이도 일부 사용자에게 노출되고 있는 상황이다.

| 기능 | AI Overviews | AI Mode |
|------|-------------|---------|
| 응답 길이 | 짧음 (3~5줄) | 길고 구조화됨 |
| 후속 질문 | 불가 | 가능 |
| 출처 표시 | 기본 | 상세 |
| 한국 지원 | 부분 | 미지원 (2026.04 기준) |


### 6. Imagen 3 업데이트 — 텍스트 렌더링이 드디어 쓸 만해졌다

이미지 생성 모델 Imagen 3에 텍스트 렌더링 정확도 개선이 이뤄졌다. AI 이미지에 글자를 넣으면 깨지거나 이상하게 나오는 문제가 고질적이었는데, 이번 업데이트로 짧은 영문 텍스트는 꽤 정확하게 나온다. 한국어 텍스트 렌더링은 여전히 불안정해서, 한글 포함 이미지는 아직 신뢰하기 어렵다.
