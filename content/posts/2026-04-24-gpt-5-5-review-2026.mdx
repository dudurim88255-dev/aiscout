## GPT-5.5란?

GPT-5.5는 OpenAI가 2026년 4월 23일 공개한 최신 대형 언어 모델이다.

수개월간 "Spud(감자)"라는 내부 코드명으로 개발 소문이 돌았고, 실제 공개 이름은 GPT-5.5로 확정됐다. 전작인 GPT-5.4를 대체하는 플래그십 모델이며, 일반용 GPT-5.5와 고정밀 작업에 특화된 GPT-5.5 Pro 두 가지 버전으로 출시됐다.

OpenAI 공동 창업자이자 사장인 Greg Brockman은 출시 발표에서 이렇게 설명했다. "이 모델의 가장 특별한 점은 더 적은 가이던스로 훨씬 많은 것을 해낸다는 점이다. 불명확한 문제를 보고 스스로 다음에 무엇을 해야 하는지 파악한다."

### 경쟁 모델과의 위치 비교

GPT-5.5가 등장하기 바로 일주일 전인 2026년 4월 16일, Anthropic은 [Claude](https://claude.ai) Opus 4.7을 공개해 공개 사용 가능한 LLM 중 성능 1위를 가져갔다. GPT-5.5는 그 자리를 다시 가져오는 데 성공했다.

특히 Terminal-Bench 2.0이라는 샌드박스 터미널 환경 작업 벤치마크에서는 비공개 모델인 [Claude](https://claude.ai) Mythos Preview마저 근소하게 앞질렀다.

Anthropic의 비공개 모델 Mythos Preview는 터미널 작업에서 82.0%를 기록했다. GPT-5.5의 82.7%는 통계적으로 사실상 동률에 가깝다. 그러나 공개 모델 중에서는 GPT-5.5가 현재 최상위라고 볼 수 있다.

OpenAI 연구 부사장 Amelia "Mia" Glaese는 이렇게 밝혔다. "벤치마크와 신뢰할 수 있는 파트너들의 피드백 모두에서, 코딩 분야의 가장 강력한 모델임이 확인된다."


## 핵심 기능 5가지

### 자율 에이전트 작업 처리 능력

GPT-5.5의 가장 큰 도약은 에이전트 성능이다.

이전 모델들은 복잡한 작업을 수행할 때 단계별로 세밀한 프롬프트를 줘야 했다. GPT-5.5는 불명확하고 다단계로 이어지는 작업을 사람의 개입 없이 스스로 처리한다. 온라인 리서치, 복잡한 코드베이스 디버깅, 문서와 스프레드시트 간의 이동 등을 자율적으로 수행한다는 점이 핵심이다.

이는 단순히 "더 스마트한 챗봇"이 아니라, 실제로 컴퓨터를 조작하고 작업을 완료하는 에이전트에 가깝다.

| 작업 유형 | 이전 GPT-5.4 방식 | GPT-5.5 방식 |
|---|---|---|
| 코드 디버깅 | 단계별 지시 필요 | 맥락 파악 후 자율 처리 |
| 온라인 리서치 | 순차적 쿼리 필요 | 멀티스텝 자율 탐색 |
| 문서 간 데이터 이동 | 반복 프롬프트 필요 | 흐름 파악 후 자동 처리 |

### Thinking 모드: 내부 검증 후 응답

GPT-5.5는 "Thinking 모드"를 [ChatGPT](https://chat.openai.com) 내에서 지원한다.

이 모드는 모델이 응답하기 전에 더 많은 내부 연산 시간을 갖고, 자신의 가정을 스스로 검증한다. 그 결과 더 정확하고 간결한 답변이 나온다. 특히 고난도 추론, 법률 리서치, 데이터 분석처럼 오류 허용치가 낮은 작업에서 효과가 두드러진다.

단순 대화에서는 일반 모드를 쓰는 것이 더 빠르고 효율적이다. Thinking 모드는 정말 복잡한 작업에 전략적으로 선택해서 쓰는 것이 좋다.

### Expert-SWE 벤치마크에서의 장기 코딩 성능

OpenAI 내부 벤치마크인 Expert-SWE는 "중앙값 인간 완료 시간이 20시간인 장기 코딩 작업"을 평가한다.

GPT-5.5는 이 벤치마크에서 GPT-5.4를 뚜렷하게 앞질렀다. 더 중요한 사실은 더 적은 토큰을 사용하면서도 더 높은 성능을 냈다는 점이다. 토큰 효율이 올라갔다는 의미이며, 비용 측면에서도 긍정적인 신호다.

20시간짜리 개발 작업을 모델 혼자 처리한다는 개념 자체가, 단순 코드 완성 도구와는 다른 차원의 접근임을 보여준다.

### NVIDIA GB200/GB300 기반 하드웨어 최적화

GPT-5.5는 NVIDIA GB200 및 GB300 NVL72 시스템 위에서 운영된다.

단순히 최신 GPU를 쓴다는 것 이상의 의미가 있다. OpenAI는 AI가 직접 작성한 맞춤형 휴리스틱 알고리즘을 활용해 GPU 코어 간 작업 분배를 최적화했다고 밝혔다. 이를 통해 토큰 생성 속도가 GPT-5.4 대비 20% 이상 향상됐다.

결과적으로 GPT-5.5는 GPT-5.4와 동일한 토큰당 지연시간을 유지하면서도 더 높은 지능 수준을 제공한다. 더 똑똑해졌지만 느려지지 않았다는 것이 핵심이다.

| 지표 | GPT-5.4 | GPT-5.5 |
|---|---|---|
| 토큰당 지연시간 | 기준값 | 동일 유지 |
| 토큰 생성 속도 | 기준값 | 20% 이상 향상 |
| 하드웨어 | 공개 미정 | NVIDIA GB200/GB300 NVL72 |
| 알고리즘 최적화 | 표준 | AI 자작성 휴리스틱 알고리즘 |

### GPT-5.5 Pro: 고난도 전문 작업 전용 버전

GPT-5.5 Pro는 일반 GPT-5.5와 별도로 설계된 고정밀 버전이다.

법률 리서치, 데이터 사이언스, 고급 비즈니스 분석처럼 정확도가 절대적으로 중요한 환경을 위해 설계됐다. 더 포괄적이고 구조화된 응답을 제공하며, 복잡한 다단계 워크플로우에서도 고품질 성능을 유지하도록 지연 최적화가 적용됐다.

현재 Pro 플랜($100~$200/월) 이상 구독자에게만 제공된다. 일반 Plus 플랜 사용자는 표준 GPT-5.5만 사용 가능하다.

- **GPT-5.5**: 범용 플래그십, Plus 플랜 이상
- **GPT-5.5 Pro**: 전문 작업 특화, 응답 구조화 강화, Pro 플랜 이상
- **공통점**: API 아직 미제공, [ChatGPT](https://chat.openai.com) 인터페이스에서만 사용 가능


## 직접 써본 솔직한 후기

### 장점 1: 프롬프트 없이도 맥락을 스스로 파악한다

기존 GPT 모델들은 "다음에 뭘 할지" 방향을 줘야 잘 따라왔다.

GPT-5.5는 달랐다. 미완성 코드베이스를 던져주고 "뭔가 이상한데 봐줘"라고만 해도, 어떤 부분이 문제인지 짚어내고 수정 방향을 제안했다. 단계별 지시 없이도 흐름을 이어갔다.

Greg Brockman이 말한 "불명확한 문제를 보고 스스로 다음 단계를 파악한다"는 설명이 과장이 아님을 체감했다.

### 장점 2: 속도가 체감될 정도로 빨라졌다

GPT-5.4와 체감 속도를 비교하면 차이가 분명히 느껴진다.

토큰 생성 속도 20% 향상은 숫자로 보면 작아 보이지만, 긴 코드나 긴 분석 결과를 받을 때 실제로 기다리는 시간이 짧아졌다. 지연시간이 동일하게 유지되면서 더 많은 내용을 더 빠르게 받는 경험이었다.

복잡한 멀티스텝 에이전트 작업에서 이 차이는 더 두드러진다.

### 장점 3: Terminal-Bench 2.0 1위라는 수치가 실제 작업에서 느껴진다

터미널 환경 작업은 가장 까다로운 AI 평가 중 하나다.

GPT-5.5의 82.7%라는 수치는 [Claude](https://claude.ai) Opus 4.7의 69.4%보다 13.3%p 높다. 이 차이는 실제로 터미널 명령어 조합, 스크립트 작성, 환경 설정 자동화 작업에서 오류 횟수로 직결된다. 비슷한 작업을 [Claude](https://claude.ai) Opus 4.7로 했을 때 막히는 구간에서 GPT-5.5는 더 자연스럽게 이어갔다.

### 아쉬운 점 1: API가 아직 없다

가장 큰 문제는 API 미제공이다.

서비스나 앱에 GPT-5.5를 직접 연동하고 싶은 개발자라면, 지금은 사용 불가다. OpenAI는 "매우 곧(very soon)" 제공 예정이라고 했지만 구체적인 날짜는 밝히지 않았다. "API 배포는 다른 안전 장치가 필요하며, 파트너 및 고객과 긴밀히 협력 중"이라는 설명이 전부다.

기업 프로덕션 환경에 바로 투입하려던 개발자에게는 치명적인 제약이다.

### 아쉬운 점 2: 유료 구독자에게만 제공된다

GPT-5.5는 [ChatGPT](https://chat.openai.com) 무료 플랜 사용자에게는 접근이 불가능하다.

Plus($20/월, ₩28,000/월) 이상 구독자만 표준 GPT-5.5를 쓸 수 있고, GPT-5.5 Pro는 Pro 플랜($100~$200/월, ₩140,000~₩280,000/월) 이상부터 접근 가능하다. 강화된 성능을 제대로 체험하려면 상당한 구독 비용이 필요하다.

경쟁사 Anthropic의 [Claude](https://claude.ai) Opus 4.7도 유료 모델이긴 하지만, 이 가격 구조는 개인 사용자에게 부담으로 작용한다.

### 아쉬운 점 3: 한국어 전용 최적화 발표 없음

GPT-5.5는 한국어로 충분히 대화 가능하지만, 공식적인 한국어 특화 최적화 발표는 없었다.

영어 코딩 환경이나 영문 문서 기반 작업에서 성능이 극대화되는 구조다. 한국어 문서 처리나 한국어 법률·분석 작업에서의 정확도는 실제 사용 전 별도로 검토하는 것이 좋다. 정확한 성능 수치는 공식 발표 확인이 필요하다.
