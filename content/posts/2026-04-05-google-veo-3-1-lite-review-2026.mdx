AI 영상 생성 툴을 쓰고 싶은데, 프리미엄 모델 요금이 부담스럽다는 생각을 해봤을 거다. 구글이 그 고민에 답처럼 내놓은 게 바로 Veo 3.1 Lite다. 고성능 Veo 3의 아래 라인업으로, '가성비 영상 AI'를 명확히 내세운다.

직접 Google AI Studio에서 테스트해봤다. 결론부터 말하면, 가격 대비 퀄리티는 납득할 만한 수준이었다. 다만 한국어 프롬프트 대응과 세밀한 모션 컨트롤은 아직 아쉬움이 남는다.


## 30초 요약

- **모델 유형:** 텍스트→영상(Text-to-Video) AI 생성 모델
- **핵심 기능:** 짧은 영상 클립 생성, Gemini API 연동, Google AI Studio 테스트 환경 제공
- **가격:** 유료 프리뷰(Paid Preview) — API 사용량 기반 과금, 무료 플랜 없음
- **대상:** 개발자, 콘텐츠 제작자, 영상 프로토타입이 필요한 스타트업
- **한국어 지원:** 프롬프트 영어 권장. 한국어 입력은 부분적으로 인식되나 결과 품질 편차 있음


## 핵심 기능 4가지


### 1. Veo 3 대비 낮은 비용 구조

Veo 3.1 Lite는 구글의 Veo 라인업 중 가장 저렴한 포지션이다. 고품질 영상이 필요하지 않은 프로토타이핑이나 반복 테스트 용도에 적합하다. 대량 생성 파이프라인을 구축하는 개발자라면 비용 부담이 크게 줄어든다.


### 2. Gemini API 직접 연동

별도 플랫폼 없이 Gemini API를 통해 바로 호출할 수 있다. 기존 Gemini 기반 앱이나 서비스에 영상 생성 기능을 추가하는 데 진입 장벽이 낮다. API 키 하나로 텍스트, 이미지, 영상 생성을 한 번에 관리할 수 있다는 점이 편하다.


### 3. Google AI Studio 무료 테스트 환경

코드 한 줄 없이 Google AI Studio에서 바로 프롬프트를 입력하고 결과물을 확인할 수 있다. 유료 API 호출 전에 품질을 직접 눈으로 확인하고 싶은 사용자에게 유용하다. 다만 AI Studio 내 테스트도 유료 프리뷰 기간에는 크레딧을 소비한다.


### 4. Veo 3.1 아키텍처 기반 품질

Lite 버전이지만 Veo 3.1 아키텍처를 공유한다. 모션 일관성과 피사체 추적 측면에서 이전 세대 모델 대비 개선된 결과를 보여준다. 다만 복잡한 장면 전환이나 긴 클립에서는 품질 저하가 눈에 띄었다.

| 비교 항목 | Veo 3 | Veo 3.1 Lite |
|---|---|---|
| 포지션 | 최고 품질 | 비용 효율 우선 |
| API 연동 | Gemini API | Gemini API |
| 가격 수준 | 높음 | 낮음 (상대적) |
| 추천 용도 | 최종 결과물 제작 | 프로토타입·반복 생성 |
| 한국어 프롬프트 | 부분 지원 | 부분 지원 |
