AI 영상 생성 도구를 쓰고 싶은데 Sora나 Runway는 월 구독료가 부담스럽다는 생각, 한 번쯤 해봤을 거다. 구글이 내놓은 Veo 3.1 Lite는 "가장 비용 효율적인 영상 생성 모델"을 내세우며 이 빈틈을 노린다.

API 호출 단가를 낮추고 Google AI Studio에서 바로 테스트할 수 있게 열어뒀다. 품질 타협이 어느 선까지인지, 한국어 프롬프트가 통하는지, 실제로 써보고 판단했다.


## 30초 요약

- **모델 유형**: 텍스트→영상, 이미지→영상 생성 AI (비디오 파운데이션 모델)
- **핵심 기능**: 저비용 고속 영상 생성, Gemini API 연동, Google AI Studio 테스트 환경 제공
- **가격**: 유료 프리뷰 (API 종량제 / 무료 플랜 없음)
- **주요 대상**: 개발자, 스타트업, 영상 콘텐츠 제작자
- **한국어 지원**: 프롬프트 한국어 입력 가능, 단 영어 대비 정확도 낮음


## 핵심 기능 4가지


### 1. 텍스트 한 줄로 영상 만들기

프롬프트를 입력하면 짧은 영상 클립을 생성한다. Veo 3.1 Lite는 Veo 3 풀 모델보다 파라미터 규모를 줄여 응답 속도를 높였다. 테스트 기준 평균 생성 시간은 약 25~40초(8초 클립 기준)였다.

움직임 표현과 조명 처리는 준수한 편이다. 단, 복잡한 동작(예: 두 사람이 악수하는 장면)에서는 손가락이 뭉개지는 현상이 여전히 나타났다.


### 2. 이미지→영상 변환 (I2V)

정지 이미지를 업로드하고 동작 설명을 추가하면 움직이는 영상으로 만들어준다. 제품 사진을 살짝 움직이는 광고 소재로 바꾸거나, 일러스트에 애니메이션 효과를 더하는 용도에 적합하다.

Lite 버전에서도 I2V 기능은 유지됐다. 다만 카메라 모션 제어 옵션은 풀 모델보다 제한적이다.


### 3. Gemini API 직접 연동

별도 플랫폼 가입 없이 기존 Gemini API 키로 바로 호출할 수 있다. 영상 생성 파이프라인을 앱에 삽입하려는 개발자에게는 진입 장벽이 낮다.

| 항목 | Veo 3.1 Lite | Veo 3 (풀 모델) |
|------|-------------|----------------|
| API 연동 | Gemini API | Gemini API |
| 예상 단가 | 상대적 저가 | 상대적 고가 |
| 생성 속도 | 빠름 (25~40초) | 보통 (40~70초) |
| 최대 해상도 | 720p | 1080p |
| 카메라 제어 | 제한적 | 풍부 |


### 4. Google AI Studio 무료 테스트

Google AI Studio에서 결제 없이 기능을 체험할 수 있다. 생성 횟수 제한이 있지만 실제 퀄리티를 확인하기에는 충분하다.

UI가 직관적이라 코드 한 줄 없이도 프롬프트를 바꿔가며 결과를 비교할 수 있다. 팀 내 비개발자 직군이 먼저 검증하기에 좋은 환경이다.

## 이런 분께 추천합니다

- **API 기반 영상 기능을 앱에 붙이려는 개발자** — Gemini API 연동이 가장 간단한 옵션 중 하나다
- **Runway·Sora의 월정액이 부담스러운 1인 크리에이터** — 쓴 만큼만 내는 구조가 맞는다
- **SNS용 짧은 제품 영상을 빠르게 뽑아야 하는 마케터** — 8~10초 클립 기준 속도가 빠르다
- **Google AI Studio로 먼저 검증하고 싶은 팀** — 코드 없이 품질 확인 가능
- **Veo 3 풀 모델 비용이 예산 초과인 스타트업** — Lite 버전으로 타협점을 찾을 수 있다
