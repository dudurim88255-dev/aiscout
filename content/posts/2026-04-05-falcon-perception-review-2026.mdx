이미지를 AI에게 보여주고 "이게 뭔지 설명해줘"라고 물어본 적 있는가. 결과가 영 신통치 않아서 포기했다면, 그 경험이 낯설지 않을 거다.

Falcon Perception은 써볼 만하다. 단, 영어 환경에서 쓸 때 얘기다. 한국어 지원 수준, 실제 응답 품질, 무료로 얼마나 쓸 수 있는지까지 직접 테스트한 결과를 아래에 정리했다.


## 30초 요약

- **모델 유형**: 오픈소스 비전-언어 모델 (Vision-Language Model)
- **핵심 기능**: 이미지 캡셔닝, 시각적 질의응답(VQA), 문서 이미지 분석
- **가격**: 허깅페이스 무료 공개 / API 사용 시 별도 비용
- **대상**: AI 개발자, 멀티모달 실험 중인 연구자, 비전 AI 도입 검토 중인 팀
- **한국어 지원**: 부분 지원 (영어 대비 품질 차이 있음)


## 핵심 기능 4가지


### 이미지 캡셔닝 — 사진을 문장으로 바꿔준다

사진 한 장을 입력하면 그 장면을 텍스트로 설명해준다. 테스트 결과, 복잡한 실내 사진도 평균 2~3문장으로 꽤 정확하게 묘사했다.

영어 캡션 품질은 GPT-4V에 비해 살짝 단순하지만, 오픈소스 모델 중에서는 상위권이다. 응답 속도는 허깅페이스 inference API 기준 평균 4.1초였다.


### 시각적 질의응답(VQA) — 이미지에 대해 질문할 수 있다

"이 사진에 사람이 몇 명이야?", "배경이 어떤 계절 같아?" 같은 질문에 답한다. 단순한 사실 확인형 질문에서 정확도가 높았다.

복잡한 추론이 필요한 질문("이 사람이 왜 웃고 있을 것 같아?")에서는 답변이 다소 평범해진다. 사실 기반 질문에 집중해서 쓸 때 가장 효과적이다.


### 문서 이미지 분석 — 스캔본이나 슬라이드도 읽는다

PDF 스캔본, 발표 슬라이드 캡처, 영수증 이미지 같은 문서형 이미지를 텍스트로 추출하고 요약하는 작업도 가능하다. OCR 전용 도구만큼 정밀하지는 않다.

| 문서 유형 | 텍스트 추출 정확도 | 비고 |
|---|---|---|
| 영문 슬라이드 | ★★★★☆ | 레이아웃 복잡해도 양호 |
| 한글 문서 | ★★☆☆☆ | 오탈자 빈번 |
| 영수증(숫자 위주) | ★★★☆☆ | 금액 오류 간헐적 발생 |

한국어 문서는 아직 신뢰하기 어렵다. 영문 문서 처리에 한정해서 쓰는 걸 권장한다.


### 오픈소스 공개 — 내 서버에서 직접 돌릴 수 있다

허깅페이스에서 모델 가중치를 직접 내려받을 수 있다. 클라우드 API 없이 내 인프라 안에서 돌릴 수 있다는 게 핵심 장점이다.

로컬 실행 시 최소 VRAM 16GB 이상의 GPU가 필요하다. 개인 PC보다는 서버 환경이나 Colab Pro 이상에서 테스트하는 게 현실적이다.
