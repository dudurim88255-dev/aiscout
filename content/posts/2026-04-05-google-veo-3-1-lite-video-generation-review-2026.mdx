영상 하나 만들려고 Runway나 Sora 구독을 고민하다가 월 요금 보고 포기한 적 있을 거다.

구글이 내놓은 Veo 3.1 Lite는 그 고민에 직접 답하는 모델이다. 가장 저렴한 영상 생성 모델을 표방하면서, Gemini API와 Google AI Studio를 통해 지금 바로 써볼 수 있다. 가격, 실제 생성 품질, 한국어 프롬프트 대응까지 직접 확인한 내용을 정리했다.


## 30초 요약

- **모델 유형:** 텍스트/이미지 → 영상 생성 AI (구글 딥마인드)
- **핵심 기능:** 저비용 영상 생성, Gemini API 연동, AI Studio 테스트 지원
- **가격:** 유료 프리뷰 (API 과금 방식, 무료 플랜 없음)
- **대상:** 개발자, 콘텐츠 제작자, AI 영상 실험 중인 마케터
- **한국어 지원:** 프롬프트 입력 가능, 단 영어 대비 품질 차이 있음


## 핵심 기능 4가지


### 1. 텍스트 한 줄로 영상 생성

프롬프트 한 줄을 입력하면 짧은 영상 클립을 만들어낸다. 기본 생성 길이는 수 초 단위다. 복잡한 편집 툴 없이 아이디어를 빠르게 시각화하는 용도로 적합하다.


### 2. Gemini API 직접 연동

별도 플랫폼 없이 Gemini API 키 하나로 연결된다. 기존에 Gemini API를 쓰던 개발자라면 추가 셋업 없이 바로 호출할 수 있다. 워크플로우에 영상 생성 기능을 끼워 넣기가 쉽다.


### 3. Google AI Studio에서 즉시 테스트

코드 한 줄 없이 Google AI Studio 웹 인터페이스에서 바로 테스트할 수 있다. 결과물을 보고 프롬프트를 수정하는 반복 작업이 빠르다. 개발 전 아이디어 검증에 적합하다.


### 4. 비용 최적화 설계

Veo 3.1 Lite는 구글 Veo 시리즈 중 가장 낮은 비용 티어를 목표로 설계됐다. 고품질 영상이 필요 없는 프로토타입, 스토리보드, 썸네일 소재 용도에 잘 맞는다. 아래 표에서 다른 모델과 간단히 비교했다.

|------|--------|-----------|-----------|

| Veo 3 | 고품질 | ★★★★☆ | 상업 영상 |

**아쉬운 점도 솔직하게 적는다.**

한국어 프롬프트는 작동하지만, 영어 프롬프트 대비 결과 품질이 떨어진다. 같은 장면 묘사를 한국어로 넣으면 구도나 분위기가 미묘하게 달라진다.

생성 길이가 짧다. 현재 공개된 클립 단위는 수 초 수준이다. 긴 영상이 필요하다면 클립을 여러 개 붙이는 후편집 작업이 필요하다.

프리뷰 기간이라 언제 요금이 바뀔지 모른다. 지금 당장 프로덕션에 붙이기보다는 테스트 목적으로 접근하는 게 낫다.
