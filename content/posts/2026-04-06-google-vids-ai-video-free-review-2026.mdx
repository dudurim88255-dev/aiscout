영상 하나 만들려고 Adobe나 Canva 유료 플랜을 결제한 경험, 한 번쯤 있을 거다. 구글이 이 시장에 무료로 끼어들었다. Vids에 Veo 3.1과 Lyria 3를 탑재해 텍스트만 입력하면 영상과 배경음악까지 자동으로 나온다.

결론부터 말하면, 무료치고는 꽤 쓸 만하다. 다만 한국어 지원과 편집 자유도에서 아직 아쉬운 부분이 있다. 이 글에서 기능, 가격, 실제 퀄리티, 그리고 추천 대상까지 정리했다.


## 30초 요약

- **모델 유형**: AI 영상 생성 + 편집 + 공유 통합 도구 (Veo 3.1, Lyria 3 기반)
- **핵심 기능**: 텍스트→영상 생성, AI 배경음악 자동 생성, 팀 공유 및 실시간 협업
- **가격**: 무료 플랜 존재 / Google Workspace 구독 시 확장 기능 제공
- **대상**: 발표 자료, 사내 교육 영상, SNS 콘텐츠를 빠르게 만들고 싶은 사람
- **한국어 지원**: 인터페이스 일부 한국어 지원, 생성 프롬프트는 영어 권장


## 핵심 기능 5가지


### 1. Veo 3.1 — 텍스트 한 줄이 영상이 된다

텍스트 프롬프트를 입력하면 Veo 3.1이 영상 클립을 자동 생성한다. 기존 Veo 2 대비 움직임 자연스러움과 해상도가 눈에 띄게 올라갔다. "회의실에서 발표하는 직장인" 같은 간단한 묘사도 제법 그럴듯하게 나온다.

무료 플랜에서도 이 기능을 쓸 수 있다는 게 핵심이다. 다만 생성 클립 길이와 해상도는 플랜에 따라 달라진다.


### 2. Lyria 3 — 영상 분위기에 맞는 음악이 자동으로 붙는다

구글 DeepMind의 음악 생성 모델 Lyria 3가 탑재됐다. 영상 내용을 분석해 어울리는 배경음악을 자동으로 만들어 붙여준다. 저작권 걱정 없이 쓸 수 있다는 점이 실용적이다.

직접 써보니 "밝고 경쾌한 느낌"이나 "진지한 프레젠테이션 분위기"처럼 분위기 지정이 가능하다. 완전히 내 취향과 맞진 않았지만, 유튜브 무료 음원보다는 자연스러웠다.


### 3. 슬라이드+영상 혼합 편집 — 구글 슬라이드가 그대로 붙는다

기존 구글 슬라이드 파일을 Vids로 바로 가져올 수 있다. 슬라이드에 AI 생성 영상 클립을 섞어 하이브리드 포맷으로 완성한다. 발표 자료를 영상으로 바꾸는 시간이 확 줄어든다.

| 편집 방식 | 기존 방식 | Google Vids |
|---|---|---|
| 슬라이드 → 영상 변환 | 수동 녹화 필요 | 자동 변환 |
| 배경음악 | 별도 편집 도구 필요 | 자동 생성 |
| 팀 공유 | 파일 전송 | 링크 공유 |


### 4. 실시간 팀 협업 — 구글 문서처럼 같이 편집한다

Google Docs처럼 여러 명이 동시에 편집할 수 있다. 댓글 달기, 버전 히스토리 확인, 링크 공유까지 구글 생태계 안에서 모두 해결된다. 이미 Google Workspace를 쓰는 팀이라면 별도 툴 없이 바로 도입 가능하다.


### 5. 나레이션 자동 생성 — 스크립트 넣으면 목소리가 나온다

슬라이드나 영상에 텍스트 스크립트를 입력하면 AI가 나레이션을 읽어준다. 목소리 톤과 속도를 조절할 수 있다. 직접 녹음하기 번거로운 상황에서 빠르게 프로토타입 영상을 만들 때 유용하다.
