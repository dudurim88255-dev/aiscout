## Google x Kaggle AI Agents Vibe Coding 코스란?

Google과 Kaggle은 2025년에 이어 2026년에도 **GenAI Intensive 시리즈**를 이어간다. 이번 에디션의 주제는 'AI 에이전트'와 'Vibe Coding'이다.

Vibe Coding은 자연어 프롬프트를 활용해 AI가 코드를 생성하도록 유도하는 개발 방식을 말한다. 기존의 라인 단위 코딩 대신 의도(intent)를 기술하면 AI가 전체 구조를 제안하는 패러다임이다. Google은 이 방법론을 AI 에이전트 개발에 접목해 5일 집중 커리큘럼으로 구성했다.

Kaggle은 전 세계 1,500만 명 이상의 데이터 과학자·ML 엔지니어가 활동하는 플랫폼이다. Google이 소유한 커뮤니티로, 실습 환경(노트북), 데이터셋, 컴피티션을 제공한다. 이번 코스 역시 Kaggle 노트북 환경 위에서 실습이 진행된다. 별도의 로컬 개발 환경 설정 없이 브라우저만으로 수강할 수 있다는 점이 경쟁 교육 플랫폼과의 가장 큰 차이다.

유사한 AI 교육 과정으로는 DeepLearning.AI의 단기 코스나 Coursera의 Google Cloud 강의가 있다. 하지만 이들은 대부분 유료이거나 수료증 취득에 비용이 발생한다. Google x Kaggle 코스는 수강 자체가 무료이며, Kaggle 커뮤니티 내 토론·피드백이 실시간으로 제공된다는 점에서 차별화된다.


## 핵심 기능 5가지

### AI 에이전트 설계 원칙 학습

이번 코스의 핵심 주제는 **AI 에이전트(AI Agents)** 다. 단순한 챗봇이 아니라 목표를 스스로 설정하고 도구를 호출해 작업을 완수하는 자율 시스템을 만드는 방법을 배운다.

에이전트 설계에서는 다음 개념들이 다뤄진다.

| 개념 | 설명 |
|---|---|
| ReAct 패턴 | 추론(Reasoning)과 행동(Acting)을 반복하는 에이전트 구조 |
| 도구 호출(Tool Use) | 에이전트가 외부 API·함수를 자율 호출하는 방식 |
| 멀티 에이전트 시스템 | 여러 에이전트가 협력해 복잡한 작업을 처리하는 구조 |
| 메모리 관리 | 장기·단기 컨텍스트를 에이전트가 관리하는 방법 |

단순 이론이 아니라 Kaggle 노트북에서 직접 구현해보는 실습 중심으로 진행된다.

### Vibe Coding 방법론 실습

Vibe Coding은 2026년 AI 개발 트렌드의 핵심 키워드 중 하나다. 자연어로 개발 의도를 설명하면 AI가 코드를 생성하고, 개발자는 이를 검토·수정하는 방식이다.

이 코스에서는 Vibe Coding을 단순 도구 사용법이 아닌 **개발 워크플로우 전체에 통합하는 방식**으로 가르친다. 구체적으로는:

- 프롬프트 작성 전략 (명확한 의도 전달법)
- AI 생성 코드 검토 및 디버깅 기법
- 에이전트 로직에 Vibe Coding 적용하는 실전 패턴

이 세 가지 흐름으로 구성된 것으로 알려졌다. 정확한 커리큘럼 세부 내용은 공식 발표 시 확인 필요.

### [Gemini](https://gemini.google.com) API 기반 실습 환경

Google이 주관하는 만큼 **[Gemini](https://gemini.google.com) API**가 실습의 중심이다. [Gemini](https://gemini.google.com) 1.5 Pro 또는 2.0 계열 모델(정확한 버전은 공식 발표 확인 필요)을 활용해 에이전트를 구현하는 실습이 포함된다.

[Gemini](https://gemini.google.com) API는 코드 생성, 멀티모달 처리, 함수 호출(Function Calling), 컨텍스트 윈도우 확장 등에서 강점을 가진다. 코스 수강생은 무료로 API 크레딧을 제공받을 가능성이 높다. 단, 공식 크레딧 제공 여부 및 한도는 발표 전이므로 확인 필요.

| 기능 | [Gemini](https://gemini.google.com) API 특징 |
|---|---|
| 함수 호출 | 에이전트 도구 연동에 최적화 |
| 멀티모달 | 텍스트·이미지·코드 동시 처리 |
| 컨텍스트 윈도우 | 긴 문서 처리에 유리 |
| 한국어 지원 | 기본 지원 (품질은 영어 대비 다소 낮음) |

### Kaggle 노트북 기반 무설치 실습

모든 실습은 **Kaggle 노트북** 위에서 진행된다. 로컬 Python 환경, GPU 설정, 패키지 설치 등 사전 준비가 필요 없다.

Kaggle 노트북은 다음과 같은 이점을 제공한다.

- 무료 GPU/TPU 컴퓨팅 리소스 (주당 사용량 제한 있음)
- 사전 설치된 ML 라이브러리 환경
- 수강생 간 노트북 공유 및 포크 기능
- 코드 실행 결과가 서버에 저장되어 언제든 재접근 가능

특히 GPU 없는 개인 PC를 사용하는 개발자에게 실질적인 진입 장벽을 낮춰주는 기능이다.

### 커뮤니티 기반 학습 지원

Google과 Kaggle은 단순 강의 제공에서 그치지 않고 **디스코드(Discord) 기반 커뮤니티**를 함께 운영한다. 이전 2025년 GenAI Intensive 코스에서도 수만 명의 수강생이 참여해 질의응답과 프로젝트 공유가 활발하게 이루어진 것으로 알려졌다.

커뮤니티 학습의 핵심 요소:

- **라이브 Q&A 세션:** Google DeepMind·Kaggle 전문가와 직접 소통
- **디스코드 채널:** 수강생 간 실시간 토론 및 디버깅 도움
- **Kaggle 토론 포럼:** 과제 풀이 접근 방식 공유
- **수료 후 네트워킹:** 글로벌 AI 커뮤니티 접근

이전 기수 수강생들의 후기에 따르면, 커뮤니티 활동이 학습 효과에 큰 영향을 미쳤다는 의견이 많았다. 정확한 수강생 수·후기 통계는 공식 발표 확인 필요.


## 직접 살펴본 솔직한 후기

공식 발표 내용과 이전 기수 수강 경험자들의 피드백을 종합해 솔직하게 정리했다.

### 장점 3가지

**1. 완전 무료 + 실무 직결 커리큘럼**

수강료가 $0이다. 그러면서도 Google DeepMind 엔지니어가 직접 설계한 커리큘럼을 사용한다. 유료 부트캠프에서 수십만 원을 내야 배울 수 있는 내용을 무료로 제공한다는 점은 명백한 강점이다.

단순 이론 강의가 아니라 Kaggle 노트북 실습 코드가 함께 제공된다. 배운 내용을 그날 바로 실행해볼 수 있는 구조다.

**2. AI 에이전트라는 시의성 있는 주제**

2026년 현재 AI 에이전트는 산업 현장에서 가장 빠르게 확산되고 있는 기술 영역이다. 채용 공고에서도 에이전트 개발 경험을 요구하는 경우가 늘고 있다. 이 코스는 그 트렌드에 정확히 맞는 주제를 다룬다.

Vibe Coding이라는 새로운 개발 방법론까지 함께 배울 수 있어, 학습의 시의성이 높다.

**3. 비동기 수강 가능 + 유연한 일정**

5일 집중 코스이지만 라이브 수업만 있는 구조가 아니다. 녹화 강의와 노트북 자료가 제공되어 본인 일정에 맞게 수강할 수 있다. 직장인이나 학생에게 현실적인 수강 환경이다.

### 단점 및 아쉬운 점

**1. 공식 강의 자료가 영어 전용**

공식 발표된 커리큘럼 자료는 전부 영어로 제공된다. 한국어 자막이나 번역본 제공 여부는 현재(2026년 4월 기준) 공식 발표 없음. 영어 기술 문서에 익숙하지 않은 입문자에게는 진입 장벽이 될 수 있다.

이전 기수에서도 한국어 지원은 없었다. 구글 번역이나 DeepL을 병행해야 하는 번거로움이 있다.

**2. 5일 집중 포맷의 학습 밀도 부담**

5일 동안 AI 에이전트 이론, Vibe Coding, [Gemini](https://gemini.google.com) API 실습, 멀티 에이전트 시스템까지 다루는 커리큘럼은 학습량이 상당하다. ML·AI 기초 지식이 없는 완전 초보자에게는 빠른 진도가 부담이 될 수 있다.

이전 기수 후기를 보면 "Day 3부터 급격히 어려워진다"는 의견이 반복적으로 등장했다. 사전에 Python 기초와 ML 개념을 익혀두는 것이 권장된다.

**3. 수료증의 공식 인정 범위 제한**

Kaggle 코스 수료증은 커뮤니티 내에서는 가시성이 높지만, 국내 기업 채용 과정에서 공식 자격증으로 인정되는 사례는 아직 제한적이다. Coursera나 edX의 공인 수료증과 동등하게 취급되지 않을 수 있다.
