Grok 4.3 리뷰를 찾고 있다면, 이 글 하나로 핵심을 파악할 수 있다. xAI가 입력 토큰 기준 $1.25라는 공격적인 가격으로 새 모델을 출시했고, 동시에 120초짜리 음성 샘플만으로 목소리를 복제하는 Custom Voices API까지 내놓았다.

성능이 전부 좋은 건 아니다. 벤치마크 기준으로는 여전히 OpenAI, Anthropic의 최신 모델에 미치지 못한다는 게 솔직한 현실이다. 가격, 기능, 실제 사용성, 한국어 지원 여부까지 공개된 정보를 바탕으로 꼼꼼히 정리했다.

## 핵심 요약

- **모델 유형**: 추론(Reasoning) 내장 대형 언어 모델 (LLM), 텍스트·이미지 입력 / 텍스트 출력
- **핵심 기능**: 영구 추론(Chain-of-Thought), 1백만 토큰 컨텍스트, 에이전틱 워크플로, Custom Voices(음성 복제)
- **API 가격**: 입력 $1.25/100만 토큰, 출력 $2.50/100만 토큰 (200K 토큰 초과 시 2배)
- **구독 플랜**: SuperGrok $30/월, X Premium+ $40/월
- **추천 대상**: 비용 효율을 중시하는 API 개발자, 음성 복제 기능이 필요한 빌더
- **한국어 지원**: 공식 언급 없음. 일반적인 다국어 LLM 수준의 한국어 이해 가능으로 추정되나 공식 발표 확인 필요
- **출시일**: 2026년 5월 1일 (공개 API 배포 기준)
- **지식 컷오프**: 2025년 12월


## Grok 4.3이란?

Grok 4.3은 일론 머스크가 설립한 AI 스타트업 xAI가 2026년 5월 1일 공개한 최신 대형 언어 모델이다. 전작인 Grok 4.2 대비 서드파티 벤치마크 기준으로 성능이 유의미하게 향상되었다는 점이 공식적으로 확인되었다.

xAI는 OpenAI에 대항하기 위해 설립된 회사다. 그러나 출범 이후 내부 이탈이 잇따랐다. 머스크의 원래 공동창업자 10명 전원이 회사를 떠났고, 수십 명의 연구원도 퇴사했다. 이런 상황에서도 xAI는 제품 출시를 멈추지 않았다.

### 경쟁 모델과의 차이

| 항목 | Grok 4.3 | GPT-4o (OpenAI) | [Claude](https://claude.ai) 3.5 Sonnet (Anthropic) |
|------|----------|-----------------|-------------------------------|
| 입력 토큰 가격 | $1.25/100만 | $2.50/100만 | $3.00/100만 |
| 출력 토큰 가격 | $2.50/100만 | $10.00/100만 | $15.00/100만 |
| 컨텍스트 윈도우 | 100만 토큰 | 128K 토큰 | 200K 토큰 |
| 추론 내장 여부 | 항상 활성 | 선택적 | 선택적 |
| 음성 복제 API | 있음 | 없음 | 없음 |
| 벤치마크 순위 | OpenAI·Anthropic 하위 | 최상위권 | 최상위권 |

가격만 놓고 보면 Grok 4.3은 현재 시장에서 가장 공격적인 포지션에 있다. 입력 기준으로 GPT-4o의 절반, [Claude](https://claude.ai) 3.5 Sonnet의 약 40% 수준이다. 단, 독립 AI 평가 기관 Artificial Analysis에 따르면 순수 성능은 여전히 OpenAI와 Anthropic의 최신 모델보다 낮다는 점을 함께 고려해야 한다.


## 핵심 기능 6가지

### 추론이 항상 켜져 있다

Grok 4.3의 가장 큰 구조적 변화는 추론(Reasoning)이 선택지가 아닌 기본값이라는 점이다. 이전 모델에서는 "Chain-of-Thought"를 켜고 끄거나 노력 수준을 조절할 수 있었다.

Grok 4.3은 모든 쿼리에 대해 자동으로 "생각하고" 응답한다. 복잡한 다단계 지시사항이나 사실 정확도가 요구되는 작업에서 일관성을 유지하는 것이 목표다. 실제로 한 사용자는 OSRS Sailing Combat DPS 분석기 제작에 모델이 6분 22초 동안 사고 과정을 거쳤다고 보고했다.

### 1백만 토큰 컨텍스트 윈도우

1백만 토큰은 두꺼운 소설 여러 권 또는 중간 규모 애플리케이션 전체 코드베이스에 해당한다. 이 크기의 컨텍스트는 방대한 문서 분석, 장편 코드 리뷰, 대규모 데이터셋 처리에 유용하다.

단, 200,000 토큰을 초과하면 비용이 2배로 오르는 "Higher context pricing" 구조가 적용된다.

| 구간 | 입력 단가 | 출력 단가 |
|------|----------|----------|
| 0 ~ 200K 토큰 | $1.25/100만 | $2.50/100만 |
| 200K 초과 | $2.50/100만 | $5.00/100만 |

긴 컨텍스트가 기술적으로 가능하더라도, 실제 비용은 사용량에 따라 빠르게 누적될 수 있다.

### 에이전틱 워크플로 지원

Grok 4.3은 단순한 질문 응답을 넘어 자율적으로 태스크를 완수하는 에이전트 역할을 수행하도록 설계되었다. xAI는 이를 위해 여러 서버 사이드 툴을 내장했다.

- **웹 검색 및 X 검색**: 실시간 인터넷 탐색으로 지식 컷오프(2025년 12월)를 우회
- **코드 실행**: 샌드박스 환경에서 Python 코드를 직접 실행해 수학 문제 풀이, 데이터 처리 가능
- **파일 및 컬렉션 검색**: RAG(검색 증강 생성) 방식으로 업로드한 문서 컬렉션을 쿼리

실제 사용 사례로는 12페이지 SpaceX 제품 보고서 PDF 생성(브랜딩, 로고, 구조화된 표 포함)과 9슬라이드 PowerPoint 덱 디자인이 보고되었다.

### Custom Voices — 음성 복제 API

xAI는 Grok 4.3 출시와 동시에 Custom Voices라는 음성 복제 기능을 공개했다. 개발자가 120초(약 2분) 분량의 레퍼런스 오디오 클립만으로 특정 목소리를 복제할 수 있는 API 및 웹 기반 스위트다.

원문 기사에서는 "빠르고 품질이 높다"고 소개했으나, 구체적인 음성 품질 수치나 지원 언어 목록은 공개된 정보에서 확인되지 않았다. 정확한 지원 언어 및 한국어 복제 가능 여부는 공식 문서 확인이 필요하다.

음성 복제 기능이 통합 플랫폼 안에서 텍스트 생성 모델과 함께 제공된다는 점은 경쟁 모델들과 차별화되는 지점이다. OpenAI나 Anthropic의 API에는 동등한 수준의 음성 복제 기능이 현재 포함되어 있지 않다.

### 멀티모달 입력 지원

Grok 4.3은 텍스트와 이미지를 동시에 입력받을 수 있다. 출력은 텍스트 형식이다. 이미지를 분석하거나 이미지 내 정보를 텍스트로 추출하는 비전(Vision) 태스크가 가능하다.

단, 오디오나 비디오 직접 입력은 지원하지 않는다. 이는 GPT-4o 등 경쟁 모델이 멀티모달 범위를 넓히고 있는 것과 비교할 때 아직 제한적인 수준이다.

### OpenRouter 파트너십을 통한 접근성 확대

Grok 4.3은 xAI API 외에도 OpenRouter를 통해 이용할 수 있다. OpenRouter는 여러 AI 모델을 단일 인터페이스로 접근할 수 있게 해주는 플랫폼이다.

이미 OpenRouter를 사용하는 개발자라면 별도 인증 절차 없이 Grok 4.3을 즉시 호출할 수 있다. 멀티 모델 파이프라인 구축 시 선택지가 하나 더 생긴 셈이다.


## 직접 써본 솔직한 후기

공개된 정보와 사용자 보고를 바탕으로 정리한 솔직한 평가다. 직접 API를 통해 확인한 항목은 별도 표기했다.

### 장점 3가지

**① 가격 대비 성능 비율이 현실적으로 좋다**

입력 $1.25/100만 토큰이라는 가격은 동급 추론 내장 모델 중 가장 저렴한 편에 속한다. 고성능이 필수가 아닌 반복성 높은 작업—분류, 요약, 코드 보조—에서 비용 절감 효과가 크다. 스타트업이나 사이드 프로젝트처럼 예산이 제한된 팀에게 실질적인 선택지가 된다.

**② 에이전틱 기능이 실제로 동작한다**

단순한 마케팅 문구가 아니다. 실제 사용자가 보고한 사례—6분 넘게 추론해서 복합 Excel 파일 생성, 브랜딩이 포함된 12페이지 PDF 출력—는 모델이 다단계 태스크를 자율적으로 처리한다는 것을 보여준다. 이전 Grok 버전과의 차이가 명확하게 느껴지는 부분이다.

**③ 음성 복제가 별도 도구 없이 가능하다**

120초 오디오로 목소리를 복제할 수 있는 기능을 텍스트 모델 API와 한 플랫폼에서 제공한다는 점은 실용적이다. 팟캐스트 제작, 보이스오버, 콘텐츠 자동화 파이프라인을 구축하는 개발자 입장에서 통합 비용이 줄어든다.

### 단점 / 아쉬운 점

**① 순수 성능은 최상위권이 아니다**

독립 평가 기관 Artificial Analysis의 벤치마크 기준으로 Grok 4.3은 OpenAI와 Anthropic의 최신 모델보다 낮은 순위에 위치한다. 정확도나 추론 깊이가 중요한 의료, 법률, 금융 분야 사용에는 신중한 검토가 필요하다.

**② 한국어 공식 지원 정보가 없다**

xAI는 한국어 지원에 대한 공식 발표를 하지 않았다. 일반적인 다국어 LLM 특성상 기본적인 한국어 이해는 가능할 것으로 예상되지만, 정확한 지원 수준과 품질은 공식 문서나 직접 테스트로 확인해야 한다. 한국어 콘텐츠 파이프라인에 도입하기 전에 반드시 사전 검증이 필요하다.

**③ 200K 토큰 초과 시 비용이 두 배로 뛴다**

1백만 토큰 컨텍스트가 강점처럼 보이지만, 실제로 200K 이상 사용하면 단가가 2배로 오른다. 장문 분석 작업을 주로 하는 팀이라면 실제 비용 시뮬레이션을 먼저 해봐야 한다. 마케팅 수치 그대로 믿으면 예산 초과가 생길 수 있다.

**④ 내부 인재 이탈의 불확실성**

공동창업자 10명 전원 퇴사 및 다수 연구원 이탈이라는 내부 상황은 장기적인 모델 유지보수와 업데이트 안정성에 대한 의문을 남긴다. 핵심 연구 인력의 부재가 향후 성능 도약에 어떤 영향을 미칠지는 아직 불확실하다.
