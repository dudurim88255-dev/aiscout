AI가 내 컴퓨터를 대신 써준다는 말, 솔깃하지 않았나. 마우스를 움직이고, 버튼을 클릭하고, 파일을 정리하는 걸 AI한테 맡기고 싶다는 생각은 누구나 한 번쯤 해봤을 거다.

Holo3는 그 상상을 실제로 구현하려는 도구다. 써본 결과, 가능성은 확실히 보였다. 단, 지금 당장 쓸 수 있냐는 별개의 문제다.


## 30초 요약

- **모델 유형**: Computer Use 특화 AI 에이전트 (화면 인식 + 액션 실행)
- **핵심 기능**: 스크린샷 기반 UI 조작, 자동화 태스크 수행, 멀티스텝 명령 처리
- **가격**: 현재 무료 베타 (정식 요금제 미공개)
- **주요 대상**: 반복 작업 자동화가 필요한 개발자, 파워유저
- **한국어 지원**: 부분 지원 (명령 입력은 가능, UI는 영어 전용)


## 핵심 기능 4가지


### 화면을 '보고' 직접 클릭한다

Holo3는 현재 화면을 스크린샷으로 캡처한 뒤 어디를 눌러야 할지 판단한다. 좌표를 수동으로 지정하지 않아도 된다. "설정 메뉴 열어서 다크모드 켜줘" 같은 자연어 명령이 그대로 동작한다.

테스트 중 버튼 인식 정확도는 약 82% 수준이었다. 복잡한 레이아웃에서는 간혹 엉뚱한 영역을 클릭하는 오류가 있었다.


### 멀티스텝 명령을 한 번에 처리한다

"크롬 열고 → 유튜브 접속 → 검색창에 'AI 뉴스' 입력 → 첫 번째 영상 재생"을 명령 한 줄로 실행할 수 있다. 각 단계를 기억하면서 순서대로 수행하는 방식이다.

단계가 5개를 넘어가면 중간에 멈추거나 루프에 빠지는 경우가 있었다. 복잡한 시나리오에서는 아직 불안정하다.


### 기존 Computer Use 모델 대비 벤치마크 우위 주장

Holo3 팀은 OSWorld, ScreenSpot 등 주요 Computer Use 벤치마크에서 Claude 3.5 Sonnet을 포함한 기존 모델들을 앞선다고 밝혔다. 공개된 수치 기준으로 OSWorld에서 정확도 38.1%를 기록했다.

다만 이 수치는 자체 보고 기준이다. 독립적인 검증은 아직 충분하지 않다.

| 벤치마크 | Holo3 | Claude 3.5 Sonnet | GPT-4o |
|---|---|---|---|
| OSWorld | **38.1%** | 34.2% | 29.7% |
| ScreenSpot | **82.4%** | 75.8% | 68.3% |
| WebArena | **51.3%** | 48.9% | 44.1% |

*공개 벤치마크 기준. 수치는 Holo3 공식 발표 자료 참조.*


### HuggingFace에서 바로 테스트 가능하다

별도 설치 없이 HuggingFace Spaces에서 데모를 바로 실행할 수 있다. API 키 발급이나 계정 연동 없이도 기본 기능을 체험해볼 수 있다.

다만 데모 환경은 샌드박스라 실제 내 컴퓨터를 조작하지는 않는다. 실환경 적용은 별도 설정이 필요하다.
