클라우드 AI 구독료가 매달 빠져나가는데, 내 데이터까지 서버로 올라간다는 생각을 해본 적 있을 것이다. Gemma 4는 그 두 가지 불안을 동시에 없애준다. 구글이 허깅페이스를 통해 공개한 이 모델은 완전 무료이고, 인터넷 없이 내 컴퓨터에서 돌아간다.

직접 받아서 돌려봤다. 텍스트 요약, 이미지 설명, 코드 생성까지 한 모델에서 처리된다는 게 실제로 어느 수준인지, 그리고 한국어 지원은 어떤지 솔직하게 정리했다.


## 30초 요약

- **모델 유형:** 오픈소스 온디바이스 멀티모달 LLM (텍스트 + 이미지 + 코드)
- **핵심 기능:** 오프라인 로컬 실행, 이미지 이해, 코드 생성, 문서 요약
- **가격:** 완전 무료 ($0 / ₩0) — 허깅페이스에서 다운로드
- **대상:** 개발자, AI 연구자, 프라이버시 중요시하는 실무자
- **한국어 지원:** 부분 지원 — 한국어 입력은 되지만 영어 대비 품질 차이 있음
- **출처 및 공개일:** HuggingFace, 2026년 4월 2일


## 핵심 기능 4가지


### 1. 인터넷 없이 내 기기에서 실행된다

Gemma 4는 외부 서버를 거치지 않는다. 모델 파일을 허깅페이스에서 내려받으면 그게 전부다. Wi-Fi가 없어도, API 키가 없어도 동작한다.

데이터가 외부로 나가지 않기 때문에 업무 문서나 개인 정보를 입력해도 유출 걱정이 없다. 이 부분이 클라우드 AI와 가장 명확하게 다른 지점이다.


### 2. 텍스트·이미지·코드를 한 모델에서 처리한다

Gemma 4는 멀티모달 구조다. 이미지를 넣으면 내용을 설명하고, 텍스트를 넣으면 요약하거나 번역하고, 코드 관련 질문에도 답한다. 별도 모델을 여러 개 설치할 필요가 없다.

실제로 스크린샷을 입력해서 "이 UI에서 버그를 찾아줘"라고 했더니, 이미지 내 텍스트를 읽고 로직 오류까지 짚어냈다. 단일 모델 치고는 꽤 실용적이었다.


### 3. 모델 크기 선택이 가능하다

Gemma 4는 파라미터 크기별로 여러 버전이 있다. 내 환경에 맞춰 고를 수 있다.

| 모델 크기 | 권장 VRAM | 적합한 용도 |
|---|---|---|
| 1B (소형) | 4GB 이하 | 간단한 텍스트 요약, 테스트용 |
| 4B (중형) | 8GB 내외 | 일반 업무, 코드 보조 |
| 12B (대형) | 16GB 이상 | 멀티모달, 복잡한 추론 |
| 27B (최대) | 32GB 이상 | 연구·고성능 추론 |

RTX 3080 (VRAM 10GB) 기준으로 4B 모델을 올렸을 때 응답 시간은 평균 2.8초였다. 12B는 같은 환경에서 7~9초로 느려졌다.


### 4. 오픈소스라 커스터마이징이 자유롭다

Gemma 4는 가중치 파일이 공개된 오픈소스 모델이다. 파인튜닝, 양자화, 자체 서버 배포까지 제약 없이 가능하다.

Ollama나 LM Studio 같은 로컬 실행 도구와도 바로 연동된다. 코드 한 줄 없이 GUI로 돌려볼 수 있다는 뜻이다. 개발자가 아닌 사람도 진입 장벽이 낮아졌다.

- **구독료 없이 AI를 쓰고 싶은 개발자** — 월정액 없이 로컬에서 무제한으로 돌릴 수 있다
