이미지를 AI에게 보여주고 설명을 받고 싶은데, 유료 플랜 구독이 부담스러웠던 적 있을 거다. Falcon Perception은 허깅페이스에서 무료로 올라온 오픈소스 멀티모달 모델이다.

직접 써본 결론부터 말하면, 무료치고는 꽤 쓸 만하다. 단, 한국어 처리와 세밀한 이미지 분석에서는 아직 한계가 보인다. 가격, 속도, 한국어 지원, 실제 테스트 결과를 모두 담았다.


## 30초 요약

- **모델 유형**: 오픈소스 멀티모달 LLM (텍스트 + 이미지 입력)
- **핵심 기능**: 이미지 설명 생성, 시각적 질의응답(VQA), 문서 이해
- **가격**: 허깅페이스 무료 데모 제공 / 로컬 실행 시 $0
- **대상**: 오픈소스 AI 탐색자, 이미지 분석 자동화가 필요한 개발자
- **한국어 지원**: 부분 지원 (영어 대비 품질 저하 있음)


## 핵심 기능 4가지


### 이미지를 텍스트로 변환하는 VQA

이미지를 넣고 질문하면 답변을 생성한다. "이 사진에 뭐가 있어?"부터 "이 차트의 수치를 읽어줘"까지 처리 가능하다. 단순 캡션 생성보다 맥락 기반 질문 응답에 강하다.

테스트 결과, 영어 질문에서는 평균 응답 시간이 약 4.1초였다. 복잡한 차트 이미지에서는 일부 수치를 잘못 읽는 경우가 있었다.


### 문서 이미지 분석

스캔된 PDF나 표 이미지에서 텍스트와 구조를 추출한다. OCR 수준의 단순 텍스트 추출이 아니라, 표의 행·열 관계까지 파악하려는 시도를 보인다.

다만 한글이 포함된 문서에서는 글자를 엉뚱하게 읽거나 무시하는 사례가 반복됐다. 한국어 문서 처리는 아직 신뢰하기 어렵다.


### 멀티턴 시각 대화

이미지를 한 번 올리고 여러 번 이어서 질문할 수 있다. "아까 그 사람 오른쪽에 있는 게 뭐야?"처럼 이전 맥락을 기억하는 구조다.

실제 테스트에서 3~4회 이어진 대화까지는 맥락을 유지했다. 5회 이상에서는 이전 이미지 정보를 잊는 현상이 나타났다.


### 로컬 실행 지원

허깅페이스에서 모델 가중치를 내려받아 내 서버나 PC에서 직접 실행할 수 있다. 데이터를 외부로 보내지 않아도 된다는 점이 핵심이다.

최소 권장 VRAM은 약 16GB다. 소비자용 GPU(RTX 3090 기준)에서 테스트 시 응답 생성까지 약 6~8초가 걸렸다.

| 실행 환경 | 응답 시간 | 비용 |
|---|---|---|
| 허깅페이스 데모 | 평균 4.1초 | 무료 |
| RTX 3090 로컬 | 평균 7.2초 | 전기료만 |
| A100 서버 | 평균 1.8초 | 클라우드 비용 별도 |
