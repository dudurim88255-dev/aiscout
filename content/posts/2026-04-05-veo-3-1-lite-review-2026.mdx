AI 영상 생성 도구를 써보고 싶은데, 요금표를 보는 순간 손이 멈춘 적 있을 거다. Sora, Runway, Pika 모두 프로 플랜은 월 수만 원을 훌쩍 넘긴다.

구글이 이 틈을 파고들었다. Veo 3.1 Lite는 "가장 비용 효율적인" 영상 생성 모델을 표방하며 2026년 3월 말 유료 프리뷰로 공개됐다. 가격 대비 품질이 실제로 납득할 수준인지, 직접 Google AI Studio에서 테스트한 결과를 정리했다.


## 30초 요약

- **모델 유형**: 텍스트→영상(Text-to-Video) AI 생성 모델
- **핵심 기능**: 프롬프트 기반 영상 생성, Gemini API 연동, Google AI Studio UI 테스트 지원
- **가격**: 유료 프리뷰(Paid Preview) — 정확한 단가는 Gemini API 요금 기준 적용
- **대상**: API 개발자, 영상 콘텐츠 제작자, 비용 민감한 스타트업
- **한국어 지원**: 한국어 프롬프트 입력 가능, UI는 영문 전용


## 핵심 기능 4가지


### 텍스트 프롬프트 한 줄로 영상 생성

글로 장면을 묘사하면 영상 클립을 만들어준다. "카페 창가에 앉아 커피를 마시는 여성, 빗소리, 따뜻한 조명" 같은 프롬프트로 수 초짜리 클립이 생성된다.

한국어 프롬프트도 인식은 됐다. 다만 영문 프롬프트 대비 장면 해석 정확도가 약 10~15% 떨어지는 느낌을 받았다. 중요한 장면은 영문으로 작성하는 걸 권장한다.


### Veo 3 대비 낮은 연산 비용

Veo 3.1 Lite는 풀 버전인 Veo 3 대비 연산량을 줄인 경량 모델이다. 구글이 공식적으로 "most cost-effective"라는 표현을 쓴 만큼 API 호출당 단가가 낮게 책정될 것으로 예상된다.

실제 영상 품질을 Veo 3와 비교했을 때 해상도, 피부 표현, 복잡한 동작 재현에서 차이가 보였다. 가벼운 홍보 클립이나 프로토타입 제작 용도라면 Lite로 충분하다.


### Gemini API 직접 연동

Veo 3.1 Lite는 Gemini API에 통합돼 있다. 개발자라면 별도 플랫폼 없이 기존 Gemini 워크플로에 영상 생성 기능을 그대로 붙일 수 있다.

| 연동 방식 | 지원 여부 |
|---|---|
| Gemini API (REST) | ✅ 지원 |
| Google AI Studio UI | ✅ 지원 |
| Vertex AI | 별도 확인 필요 |
| 서드파티 플러그인 | ❌ 미지원 |


### Google AI Studio에서 UI 테스트 가능

코드를 짜지 않아도 된다. Google AI Studio 웹 인터페이스에서 프롬프트를 입력하고 영상을 바로 확인할 수 있다.

단, 현재는 유료 프리뷰 단계다. 무료 계정으로는 접근이 제한되며, 구글 클라우드 결제 수단이 연결돼 있어야 테스트가 가능하다. 진입 장벽이 아예 없는 건 아니다.

## 이런 분께 추천합니다

- **Gemini API를 이미 쓰고 있는 개발자** — 별도 플랫폼 없이 영상 기능을 추가할 수 있다
- **Runway, Pika 요금이 부담스러운 소규모 팀** — 경량 모델이라 단가가 낮게 형성될 가능성이 높다
- **프로토타입이나 시안용 영상이 필요한 기획자** — 완성도보다 속도와 비용이 우선인 상황에 맞다
- **구글 생태계(Google Cloud, Workspace) 중심으로 작업하는 팀** — 연동 피로도가 낮다
- **AI 영상 생성 트렌드를 저비용으로 먼저 경험해보고 싶은 사람** — 풀 버전 전에 Lite로 감을 잡을 수 있다

반면 Gemini API 연동이라는 포지션은 명확하다. 이미 구글 AI를 쓰고 있다면 추가 비용 없이 영상 기능을 붙일 수 있다는 건 실질적인 장점이다.
