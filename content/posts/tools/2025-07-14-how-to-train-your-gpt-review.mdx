# how-to-train-your-gpt: LLM을 밑바닥부터 만드는 12챕터 튜토리얼

> 토크나이저부터 어텐션, 학습 루프, 추론 엔진까지—3,671줄 전량 주석 처리된 LLM 구현 교재.

## 한 줄 요약

how-to-train-your-gpt는 LLaMA 3 스타일의 현대적 언어 모델을 밑바닥부터 직접 구현하며 학습할 수 있도록 설계된 12챕터짜리 인터랙티브 코드 교재다. 파이썬 기초만 알면 시작할 수 있고, 모든 코드 라인에 "무엇을 하는지"와 "왜 필요한지"가 주석으로 달려 있다.

## 무엇을 해결하나

대부분의 ML 튜토리얼은 두 극단 중 하나에 빠진다. `model = GPT().fit(data)` 수준으로 끝나거나, 40페이지짜리 논문을 전제로 한다. 이 레포는 그 중간 공백을 정확히 채운다. API 호출법이 아니라 `1/√d_k`의 분산 논증, RoPE가 상대 위치를 회전으로 포착하는 원리, pre-norm이 deep 네트워크에서 유리한 이유까지 코드와 비유를 동시에 제공한다.

## 핵심 기능 3개

**1. 100% 주석 커버리지**
3,671줄 전체에 주석이 달려 있다. 단순 설명이 아니라 "5살에게 설명하듯" 비유를 곁들이고, 바로 뒤에 엔지니어 수준의 구현이 따른다. Attention 메커니즘을 파티 비유와 실제 숫자 워크스루로 동시에 다룬다.

**2. 12챕터 체계적 커리큘럼**
Overview → Setup → Tokenization(BPE) → Embeddings → Attention → Transformer Block → Training Loop → Inference까지 순서대로 진행된다. 각 챕터가 독립 마크다운 파일이라 필요한 부분만 골라 읽기에도 좋다.

**3. LLaMA 3 스타일 아키텍처**
학습용이라고 구식 모델을 쓰지 않는다. RMSNorm, RoPE, Grouped-Query Attention 등 최신 LLM에 실제로 쓰이는 기법을 직접 구현한다. ChatGPT·Claude·Mistral과 동일 계열 아키텍처를 손으로 만들어 보는 셈이다.

## 시작 방법

```bash
git clone https://github.com/raiyanyahya/how-to-train-your-gpt.git
cd how-to-train-your-gpt
python -m venv venv
source venv/bin/activate
pip install torch
```

이후 `chapters/00_overview.md`부터 순서대로 읽으며 코드를 직접 실행하면 된다. GPU가 없어도 CPU 환경에서 학습 가능하도록 설계되어 있다.

## 누구에게 유용한가

- **파이썬 개발자**: ChatGPT가 실제로 어떻게 동작하는지 궁금하지만 ML 경험이 없는 경우.
- **CS 학생**: Transformer 논문을 읽다 막혔을 때 실제 숫자가 흐르는 예제가 필요한 경우.
- **LLM 아키텍처를 평가해야 하는 엔지니어**: RoPE vs Learned Position, RMSNorm vs LayerNorm 같은 트레이드오프를 코드 레벨에서 비교하고 싶은 경우.

## 한계

- 프로덕션 수준의 학습 파이프라인(분산 학습, FSDP, 대규모 데이터 처리)은 다루지 않는다.
- 스타 235개 수준의 초기 프로젝트라 커뮤니티 피드백과 이슈 대응이 아직 얇다.
- 한국어 문서는 제공되지 않으며, 영어 주석만 존재한다.
- 실제 대형 모델 학습이 아닌 교육 목적 코드이므로, 성능 최적화나 스케일링 관점의 내용은 기대하기 어렵다.
