# Future AGI – LLM 에이전트의 환각을 추적하고 고치는 올인원 플랫폼

Future AGI는 LLM 및 AI 에이전트 애플리케이션을 평가(Evals)·관측(Observability)·개선하는 오픈소스 엔드투엔드 플랫폼이다. 트레이싱, 시뮬레이션, 데이터셋 관리, AI 게이트웨이, 가드레일까지 하나의 피드백 루프 안에서 돌아간다. Apache 2.0 라이선스이며 셀프호스팅을 지원한다.

## 무엇을 해결하나

AI 에이전트를 프로덕션에 올리면 환각, 비용 폭증, 품질 편차 같은 문제가 동시에 터진다. 보통은 트레이싱 도구 따로, 평가 프레임워크 따로, 게이트웨이 따로 붙여야 한다. 도구가 분산되면 피드백 사이클이 느려지고, 문제 원인을 역추적하기 어렵다. Future AGI는 이 파편화를 하나의 플랫폼으로 통합해 "관측 → 평가 → 개선" 루프를 단축시키는 데 초점을 맞춘다.

## 핵심 기능 3가지

**1. 트레이싱 + 평가 통합**
스팬(span) 단위로 LLM 호출을 추적하고, 해당 스팬에 곧바로 평가(eval)를 실행할 수 있다. 별도 파이프라인을 구성할 필요 없이 트레이스 뷰에서 바로 점수를 확인한다.

**2. 시뮬레이션**
음성 에이전트 대화 등 실제 사용 시나리오를 시뮬레이션으로 재현한다. 엣지 케이스를 프로덕션 트래픽 없이 테스트할 수 있어 배포 전 검증 단계를 강화한다.

**3. 게이트웨이 + 가드레일**
AI 게이트웨이를 통해 비용·지연 시간 대시보드를 제공하고, 가드레일로 유해 출력이나 정책 위반을 실시간 차단한다. 모니터링과 제어가 같은 화면에 있다.

## 설치 및 사용 예시

Python SDK는 PyPI에서 바로 설치할 수 있다.

```bash
pip install ai-evaluation
```

셀프호스팅은 레포를 클론한 뒤 Docker Compose로 올린다.

```bash
git clone https://github.com/future-agi/future-agi.git
cd future-agi
docker compose up -d
```

JavaScript/TypeScript 환경이라면 npm 패키지를 사용한다.

```bash
npm install @traceai/fi-core
```

클라우드 버전은 별도 설치 없이 [app.futureagi.com](https://app.futureagi.com/auth/jwt/register)에서 무료로 시작할 수 있다.

## 누구에게 유용한가

- LLM 기반 에이전트를 프로덕션에 운영 중이거나 준비 중인 팀
- 평가·트레이싱·게이트웨이를 각각 다른 SaaS로 쓰는 게 비효율적이라 느끼는 개발자
- 셀프호스팅이 필수 조건인 보안·컴플라이언스 환경

## 한계

현재 Nightly 릴리스 단계로, README 자체에도 "rough edges를 예상하라"고 명시되어 있다. 안정 버전이 아직 나오지 않았으므로 프로덕션 크리티컬 워크로드에 바로 도입하기엔 리스크가 있다. 커뮤니티 규모(Stars 252)도 초기 단계라 이슈 대응 속도나 플러그인 생태계가 성숙하지 않다. 경쟁 도구인 LangSmith, Arize Phoenix 등과 비교하면 문서화와 통합 예제가 부족한 편이다. 다만 Apache 2.0 라이선스와 셀프호스팅 지원은 명확한 차별점이므로, 안정 버전 출시 이후 재평가할 가치가 충분하다.
