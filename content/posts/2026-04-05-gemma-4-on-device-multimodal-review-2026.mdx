클라우드 AI에 매달 구독료를 내면서 "이게 꼭 필요한가" 싶었던 적 있을 거다. Gemma 4는 인터넷 없이 내 기기에서 직접 돌아가는 구글의 오픈소스 멀티모달 AI다. 무료로 쓸 수 있고, 이미지와 텍스트를 동시에 이해한다.

이번 글에서는 Gemma 4를 허깅페이스에서 받아 직접 실행해봤다. 응답 속도, 한국어 품질, 이미지 이해 능력까지 솔직하게 정리했다.


## 30초 요약

- **모델 유형**: 온디바이스 실행 가능한 멀티모달 오픈소스 LLM
- **핵심 기능**: 이미지+텍스트 동시 이해, 로컬 실행, 경량 추론
- **가격**: 완전 무료 (오픈소스, Apache 2.0 라이선스)
- **대상**: 개발자, AI 연구자, 프라이버시 중시 사용자
- **한국어 지원**: 가능하나 영어 대비 품질 낮음 (체감 80% 수준)


## 핵심 기능 5가지


### 1. 스마트폰에서도 돌아가는 경량 설계

Gemma 4는 모바일 기기 수준의 하드웨어에서 실행되도록 설계됐다. 4비트 양자화 버전 기준, 8GB RAM 환경에서도 구동이 가능했다. 클라우드 없이 내 기기 안에서 추론이 끝난다는 건 프라이버시 측면에서 확실한 강점이다.

실제로 M2 맥북에어에서 4B 모델을 돌렸을 때 첫 토큰까지 걸린 시간은 평균 **1.8초**였다. 이후 토큰 생성 속도는 초당 약 **28토큰** 수준이었다.


### 2. 이미지와 텍스트를 함께 처리하는 멀티모달

텍스트만 받던 이전 Gemma 시리즈와 달리, Gemma 4는 이미지를 입력으로 받을 수 있다. 사진을 주고 "이 음식 칼로리 대략 얼마야?"처럼 묻는 게 가능하다. 이미지 설명, 차트 분석, 문서 OCR 보조 등 활용 범위가 넓어졌다.

| 테스트 항목 | 결과 |
|---|---|
| 음식 사진 → 재료 추측 | 주요 재료 5개 중 4개 정확 |
| 차트 이미지 → 수치 해석 | 막대 그래프 기준 3개 중 2개 정확 |
| 한국어 손글씨 → 텍스트 변환 | 인식률 약 60% (아직 불안정) |
| 영문 문서 스캔 → 요약 | 핵심 내용 정확하게 요약 |


### 3. 허깅페이스에서 즉시 내려받기

별도 가입이나 승인 없이 허깅페이스에서 바로 다운로드할 수 있다. `google/gemma-4` 리포지토리에서 모델 파일을 받으면 된다. Ollama, LM Studio, 로컬 파이썬 환경 모두 호환된다.

```bash
# Ollama 기준 실행 예시
ollama pull gemma4
ollama run gemma4
```

설치부터 첫 응답까지 걸린 시간은 약 **7분**(4B 모델, 100Mbps 환경 기준)이었다.


### 4. 긴 문맥을 한 번에 처리하는 128K 컨텍스트

Gemma 4는 최대 **128,000 토큰**의 컨텍스트 윈도우를 지원한다. 긴 PDF, 코드베이스, 소설 전체를 한 번에 넣고 질문할 수 있다. 로컬 모델 중 이 수준의 컨텍스트를 무료로 지원하는 건 드물다.

단, 컨텍스트가 길어질수록 로컬 환경에서 메모리 사용량이 급격히 늘어난다. 32K 이상에서는 16GB RAM 이상 권장이다.


### 5. Apache 2.0 라이선스 — 상업적 사용 가능

Gemma 4는 Apache 2.0 라이선스로 배포된다. 개인 프로젝트는 물론 상업용 서비스에도 무료로 활용할 수 있다. 코드에 포함하거나, API 서버로 래핑해서 제품에 넣는 것 모두 허용된다.

이전 Gemma 버전은 사용 제한 조건이 있었는데, 이번에 완전히 풀렸다. 스타트업이나 개인 개발자에게 실질적인 이점이다.
