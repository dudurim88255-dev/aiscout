산림 파괴 데이터를 찾으려면 보통 여러 기관 사이트를 뒤져야 하고, 최신 정보인지도 확신하기 어렵다. 구글이 브라질 정부와 협력해 만든 이 위성 지도는 그 번거로움을 상당 부분 줄여준다.

결론부터 말하면, 전문 연구자가 아니어도 쓸 수 있을 만큼 직관적이다. 이 글에서는 핵심 기능, 실제 데이터 품질, 한국에서의 활용 가능성까지 정리했다.


## 30초 요약

- **모델 유형**: 위성 이미지 + AI 변화 감지 (Google Earth Engine 기반)
- **핵심 기능**: 브라질 산림 파괴 실시간 감지, 시계열 비교, 지역별 데이터 시각화
- **가격**: 공공 데이터 기반, 현재 무료 접근 가능 (API 사용량 따라 달라질 수 있음)
- **주요 대상**: 환경 연구자, 언론인, 정책 입안자, 기후 관련 NGO
- **한국어 지원**: 인터페이스 한국어 미지원, 데이터 자체는 언어 무관


## 핵심 기능 5가지


### 1. AI 기반 산림 변화 자동 감지

위성 이미지만 쌓아두는 게 아니다. AI가 이전 이미지와 현재 이미지를 비교해 변화 지점을 자동으로 표시한다. 벌목, 화재 흔적, 농지 개간 등을 구분해서 색상으로 나눠 보여준다.

감지 정확도는 픽셀 단위 수준이다. 공식 발표 기준 10m 해상도 이상의 이미지를 사용한다.


### 2. 시계열 before/after 비교 기능

특정 지역을 선택하면 과거와 현재 이미지를 슬라이더로 비교할 수 있다. 몇 년 사이 얼마나 많은 산림이 사라졌는지 시각적으로 바로 확인된다.

이 기능이 특히 유용한 건 언론 보도나 연구 발표 자료를 만들 때다. 스크린샷 한 장으로 변화를 설명할 수 있다.


### 3. 브라질 정부 공식 데이터 연동

단순히 구글 단독 프로젝트가 아니다. 브라질 정부 산림 감시 시스템(PRODES, DETER)과 데이터를 연동한다. 공식 통계와 위성 이미지가 같은 화면에서 확인된다.

이건 신뢰도 면에서 꽤 중요한 차이다. 민간 추정치가 아니라 정부 공인 데이터를 쓴다는 의미다.

| 연동 데이터 출처 | 제공 정보 | 업데이트 주기 |
|---|---|---|
| PRODES | 연간 산림 파괴 면적 | 연 1회 |
| DETER | 실시간 경보 데이터 | 일 단위 |
| Google Earth Engine | 위성 원본 이미지 | 수일 내 |


### 4. 지역별 필터링 및 면적 계산

주(州)별, 보호구역별로 필터를 걸어 원하는 지역만 볼 수 있다. 선택한 구역의 면적과 파괴 비율을 수치로 바로 보여준다.

아마존 전체를 보다가 특정 원주민 보호구역 안의 상황만 뽑아보는 것도 클릭 몇 번으로 된다.


### 5. 데이터 내보내기 및 API 접근

연구 목적이라면 데이터를 직접 다운로드하거나 Google Earth Engine API로 가져올 수 있다. GeoTIFF, CSV 등 일반적인 포맷을 지원한다.

다만 API를 쓰려면 Google Cloud 계정이 필요하고, 대량 요청 시 비용이 발생할 수 있다. 소규모 분석은 무료 할당량 안에서 가능하다.
